# Scorecard Mapping Guide

The app preserves the established scorecard design and places information into equivalent Excel and PDF locations.

## Excel cell mapping

| App area | Workbook cell |
|---|---:|
| Top game line | A2 |
| Weather, umpires, and broadcast block | A3 |
| Replay title | J3 |
| Away replay line | J4 |
| Home replay line | M4 |
| Away scoreboard label | A6 |
| Home scoreboard label | A7 |
| Away lineup title | A9 |
| Home lineup title | A21 |
| Away pitching title | A33 |
| Home pitching title | A41 |
| Game notes | J33 |

## Excel list areas

| Section | Start cell | Rows |
|---|---:|---:|
| Away lineup | A11 | 9 |
| Home lineup | A23 | 9 |
| Away pitchers | A35 | 6 |
| Home pitchers | A43 | 6 |

## PDF layout

The PDF uses a high-resolution image of the actual blank scorecard as its background. Text is placed over the same game-information, lineup, pitching, replay, and notes areas.

The PDF page is fixed at:

- 612 x 792 points
- US Letter portrait
- One page
- 18-point outer margin, equal to 0.25 inch

Long entries are reduced to fit their designated space. Game notes wrap within the notes box and are reduced when necessary.

## Preserved areas

- Scoreboard grid
- At-bat plus signs and counter graphics
- Borders and alternating row shading
- Pitching statistic columns
- Replay and extra-inning boxes
- Game notes area
- One-page Letter layout

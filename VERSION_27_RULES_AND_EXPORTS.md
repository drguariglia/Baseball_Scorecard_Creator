# Version 27 controlling specification

1. Regulation ending: home lead after top 9 ends the game; away lead after completed bottom 9 ends the game; a home lead created in bottom 9 or any later bottom half is an immediate walk-off.
2. Regular-season extras: beginning in top 10, each half-inning starts with the batting-order player immediately preceding the leadoff hitter on second base. Existing substitution controls permit a pinch-runner.
3. Postseason/traditional option: extra innings begin with empty bases.
4. Continuation PDF: innings 1–10 remain on page 1; any inning 11 or later adds page 2 covering innings 11–20 while retaining all data and cumulative totals.
5. Safe export: Game File, Excel, and PDF downloads never clear data automatically. A post-export dialog defaults to Keep Game Open; Clear Card requires a second confirmation.
6. Version 26.1 features remain: chronological Game Notes, Key Play, substitutions, pitching changes, individual errors, HR-only scoring notation, ABS challenges, undo, and reset controls.

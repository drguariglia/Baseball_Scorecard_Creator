# Version 27.2 Pitcher Export Correction

The PDF and Excel pitching sections now contain only pitchers who actually entered the game, in their first-appearance order.

- The starting pitcher appears first after recording game activity.
- Each selected reliever follows in the order entered.
- Unused active-roster pitchers are not printed.
- Before the game begins, only the currently selected starter is printed.
- Each printed pitcher retains uniform number, name, throwing hand, record, ERA, and strikeouts.
- The six-row printed scorecard limit is unchanged.

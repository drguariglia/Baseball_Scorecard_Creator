# Version 32 Plate-Appearance Result-Only Display

Version 32 remains the controlling version.

## Change

- Plate-appearance scoring boxes now show only the recorded outcome and essential play notation.
- Run totals no longer appear inside plate-appearance boxes.
- RBI totals remain omitted from plate-appearance boxes.
- Outcome details such as `HR`, `K WP`, `K PB`, fielding-error notation, `SB2`, `CS3`, and `DIH` remain visible because they describe what happened on the play.
- Runs and RBI continue to be calculated, saved, edited, summarized, and exported in their dedicated statistical columns.

## Compatibility

- No saved-game structure changed.
- No scoring calculation changed.
- Existing Version 32 games require no migration.

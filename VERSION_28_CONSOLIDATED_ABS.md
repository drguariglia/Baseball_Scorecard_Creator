# Version 28 — Consolidated ABS Challenge Tracking

The separate ABS challenge section was moved into the broadcast-style Live Game Center.

## Changes

- The away and home challenge balances now appear directly beneath the score, game situation, batter, pitcher, on-deck, and in-the-hole information.
- Each team retains its challenge tokens, available count, status, and Record Challenge control.
- The challenge history is contained in a compact expandable Challenge Log.
- All existing challenge rules, editing, extra-inning grants, PDF/Excel tracking, saved-game data, and autosave behavior remain unchanged.
- Desktop and mobile layouts use one unified scoring panel instead of a second full-width ABS card.

# Version 30 iPhone Date Field Fix

The startup Setup screen date inputs are constrained to the available mobile card width.

- The scheduled-game date and manual game date use full-width, zero-minimum sizing.
- iPhone Safari native date-control content is constrained inside the input boundary.
- Mobile date inputs use a 16-pixel font to prevent automatic page zoom.
- Parent labels clip native-control paint overflow without changing desktop layout.
- The app remains Version 30.

# Version 28 — Broadcast-Style Live Game Center

Version 28 uses the uploaded television scorebox image as a structural reference for a more streamlined Live Scoring environment without copying team logos or network branding.

## Live Game Center

- The former standalone scoreboard and oversized count have been consolidated inside Pitch-by-Pitch Scoring.
- Away and home teams appear in stacked rows with dynamic abbreviations, full names, and scores.
- The inning indicator uses an up triangle for the top half and a down triangle for the bottom half.
- Outs, game status, base occupancy, and count remain visible together.
- Balls and strikes use a compact count sized at roughly half the previous display.
- Current batter and pitcher details are displayed in the scorebox information band.
- On-deck and in-the-hole batters update from the active batting order and honor substitutions.
- The scorebox is responsive for desktop, tablet, and iPhone use.

## Compatibility

All Version 27.2 behavior remains: continuous autosave, protected iPhone roster persistence, MLB regulation and extra-inning rules, two-page extra-inning PDFs, roster and pitcher substitutions, errors, Key Plays, chronological Game Notes, ABS challenges, pitch tracking, Excel/PDF exports, and actual pitcher order of appearance.

Version 27.2 current and backup browser snapshots are migrated automatically into the Version 28 storage keys.

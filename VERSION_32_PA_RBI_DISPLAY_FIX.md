# Version 32 Plate-Appearance RBI Display Correction

Version 32 remains the controlling version.

## Change

- Removed repeated RBI text from every plate-appearance scoring box in the live scorecard, print preview, Excel inning cells, and PDF scorecard.
- RBI values continue to be calculated from each recorded play.
- RBI totals continue to populate the dedicated RBI column beside each batter.
- Run totals are also omitted from plate-appearance boxes; the boxes now contain only the outcome and essential play detail.
- Home-run notation remains the compact `HR` code.

## Compatibility

- No saved-game format changed.
- No scoring calculations changed.
- No existing Version 32 game data needs migration.
